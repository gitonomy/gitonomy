Gitonomy TwigBundle
===================

GitonomyTwigBundle gives you ability to easily render blocks about git:

* File diff
* Commit informations
* Graph logs
* ...

Installation
------------

* `Install the bundle in a Symfony2 application <install/symfony2.rst>`_
* `Install in a boilerplate PHP application <install/raw.rst>`_
